##🇳🇵 Nepali News RAG System

A Retrieval-Augmented Generation (RAG) system for Nepali news that enables semantic search and question answering using embeddings and vector search.

##🚀 Overview

This project builds an end-to-end pipeline that:

Scrapes Nepali news articles
Splits them into meaningful chunks
Converts text into embeddings using sentence-transformers
Stores vectors in FAISS for fast similarity search
Retrieves relevant context for user queries
